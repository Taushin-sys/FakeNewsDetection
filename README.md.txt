# Fake News Detection

This project uses Python and Machine Learning to detect whether news is real or fake.

## Dataset
- Sourced from Kaggle: Fake and Real News Dataset
- Files: `Fake.csv` and `True.csv`

## Tools Used
- Python, Pandas, Scikit-learn
- Logistic Regression Model

## Accuracy
Achieved 94.63% accuracy.
